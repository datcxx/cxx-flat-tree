# SYNOPSIS
A series of functions to map a binary tree to a list.

# USAGE

```cpp
#include "./flat-tree.hxx"
```

# EXAMPLE
```cpp
flatTree::Iterator it;

cout << it.parent() << endl;
```

## See also
